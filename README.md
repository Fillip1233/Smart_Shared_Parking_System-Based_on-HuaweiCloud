## 项目介绍
本套智慧共享停车系统是一个基于物联网技术，创新研发出集预约，共享，管理停车三大功能的一套新科技系统。它能够将闲置的车位利用共享出来，解决城市不同区域停车位昼夜利用率差别大，停车位管理困难以及停车体验不佳等问题，为城市停车提供崭新的解决方案。
